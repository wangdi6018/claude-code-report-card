The 6 Levels of Claude Code

youtu.be/TUKYbUIXLOE

Claude Code 활용 능력을 6단계로 나눈 가이드. 1단계부터 차근차근 쌓아 올라가는 구조 — 단순히 일 시키기에서 시작해서, 나중엔 여러 Claude를 팀처럼 굴리는 수준까지 발전해요.

  1. Lv 1Prompt Engineer
    Claude한테 일을 정확하게 시킬 줄 알고, 결과가 맞는지 비판적으로 확인하기.
  2. Lv 2Planner
    코딩 들어가기 전에 "이렇게 갈게요?" 합의부터 하기. 반박도 적극적으로.
  3. Lv 3Context Engineer
    Claude의 단기 기억(컨텍스트)이 헷갈리지 않게 자료 양 조절하기.
  4. Lv 4Integrations
    Claude한테 슬랙·DB·피그마 같은 외부 도구 연결해서 같이 쓰게 하기.
  5. Lv 5Workflow Opt
    자주 하는 반복 작업을 매뉴얼(=스킬)로 만들어 자동화하기.
  6. Lv 6Scaling
    Claude 한 명이 아니라 여러 명을 동시에 굴리기 (팀으로 확장).
💡 Lv 1~6은 "어떤 능력이 있는가" 카테고리이고, 아래 점수의 A·B·C·D는 "얼마나 잘 쓰는가" 등급이에요. 다른 축이에요.
